# Video.js Skill

## Overview

Este skill contiene toda la documentación sobre **Video.js**, la librería open-source para la construcción de reproductores HTML5/Video. Permite acceder su sintaxis y sus guías de configuración avanzadas, abarcando desde integraciones básicas hasta manejo especializado y plugins.

## Tareas Categóricas

### 1. API Exhaustiva

Si necesitas consultar las minucias, métodos globales, estáticos o propiedades del ciclo de vida del propio framework puedes encontrar los detalles en la referencia completa que se encuentra en `references/docs.md`. Esta referencia te guiará en el uso avanzado de clases del ecosistema como `Player`, `Component` o el utilitario `browser`.

### 2. Guías y Conceptos Clave

Si en cambio requieres aprender a instalarlo o un "Cómo hacer", por ejemplo cómo configurar plugins, cómo manejar skins, o cómo lograr la migración desde la Versión 7, lee nuestra compilación orientada a tutoriales y guías disponible en `references/guide.md`.
Este documento incluye la Referencia de Opciones fundamental para instanciar el componente `<video>`.

## Referencias

- `references/docs.md`: Contiene la documentación pura en formato markdown del API autogenerada (JSDoc), con los métodos, clases y tipos de parámetros. (Más de 150.000 líneas).
- `references/guide.md`: Contiene los artículos instructivos, la hoja de configuración paso a paso (**Opciones de Setup**) y best-practices sugeridas por Video.js.
